## Setup
This repository uses Git LFS for sheet music PDFs and scans.

Install Git LFS first:
- macOS: `brew install git-lfs`
- Windows: https://git-lfs.com
- Linux: `sudo apt install git-lfs` (or your package manager)

Then run once:
```bash
git lfs install
git clone https://github.com/rdrew/Rob_Drew_Sheet_Music_DB.git
